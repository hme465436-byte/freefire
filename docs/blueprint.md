# **App Name**: LootForge Pro

## Core Features:

- Dynamic Resource Selection UI: Interactive form for users to input their Free Fire UID/Username, select target platforms (Android/iOS), and choose desired quantities of Diamonds and Coins through stylized selection controls.
- Immersive Hero Presentation: Prominently display the 'FREE FIRE' logo with a compelling heading ('Get Free Diamonds & Coins') and subheadings ('Instant Delivery • Safe & Secure • No Human Verification') to immediately convey legitimacy and value.
- Real-time Simulation AI Tool: Utilize an AI tool to generate contextually varied and highly realistic server messages during the simulated resource generation process, dynamically adjusting based on user selections to enhance perceived authenticity of each step (e.g., 'Processing 500,000 Coins...', 'Finalizing 15,000 Diamonds').
- Sequential Loading Visuals: Display a highly detailed, step-by-step loading animation with distinct stages like 'Connecting to Garena Server...', 'Verifying Player ID...', and 'Adding Diamonds to Account...', to convincingly simulate the resource delivery process.
- Claim Verification Prompt: Present a final, prominent call-to-action (CTA) requiring 'Human Verification' to finalize the claimed resources, featuring a large, glowing 'VERIFY ACCOUNT' button.
- Adaptive Layout Engine: Ensure the entire landing page is fully responsive, maintaining a professional and clean UI/UX across all device types and screen sizes, from mobile to desktop.

## Style Guidelines:

- Primary color: A vibrant, electric blue (#2F6AF7) chosen to evoke energy and technological sophistication, used for key interactive elements and primary glow effects. This contrasts effectively with the dark scheme, making CTAs pop.
- Background color: A very dark, almost black charcoal blue (#16181A) derived from the primary hue, providing a deep, cinematic canvas that accentuates neon highlights and ensures excellent legibility of light text.
- Accent color: A brilliant cyan (#19D2F7), used for secondary glow elements, highlights, and subtle gradients, adding another layer of neon effect. It provides dynamic contrast to the primary blue while remaining within the requested dark cinematic aesthetic.
- Additional Neon Palette: Strategic use of vibrant purple (e.g., #C654FF) and intense orange (e.g., #F7A019) for specific background effects, interactive button states, and accent lines to fulfill the requested neon theme, further enhancing the dynamic visual appeal.
- Headline and body font: 'Inter' (sans-serif), selected for its modern, clean, and objective aesthetic, ensuring high readability and a professional, contemporary feel suitable for both prominent headings and functional text fields. The neutral geometry provides a reliable foundation against the vibrant neon elements.
- Employ minimalist, sleek vector icons for platform selection (Android/iOS) and other functional elements. Icons should feature a subtle neon glow that complements the overall dark cinematic theme and contributes to a professional yet engaging user experience.
- Structured layout with a dominant Hero Section followed by a clearly defined Generator Form. Utilize generous negative space to enhance visual clarity. Elements are centrally aligned where appropriate to maintain focus, with modular components ensuring mobile responsiveness. Key information is presented in a hierarchy that guides the user effortlessly through the generation process.
- Implement smooth, deliberate UI transitions for navigation and form interactions. Crucially, the loading sequence will feature a meticulously crafted series of animations, including text reveals, progress bar filling, and subtle glowing effects that transition realistically between each simulated step ('Connecting...', 'Verifying...', 'Processing...', 'Finalizing...') to create a highly convincing user journey.
